# Set-up

Very bad jklm.fun bot made withi puppeteer and typescript.

run `tsc` to build

run the following:
`npm install an-array-of-english-words`
`npm install prompt`
`npm install puppeteer`

# Start
1. `cd dist`
2. `node index.js`
3. Paste your link
4. have fun
